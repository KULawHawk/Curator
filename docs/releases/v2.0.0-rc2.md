## What v2.0.0-rc2 is

The first soak-period bugfix on top of v2.0.0-rc1. A wire-format regression in the MCP tool surface — broke 6 of 9 tools the moment any of them had data to return — surfaced under interactive testing against the running server. Fixed at the pydantic model layer with a dedicated `UTCDatetime` field type that coerces naive datetimes to aware UTC and forces the explicit `+00:00` offset on the wire.

Also subsumes an in-flight uncommitted fix to `src/curator/__init__.py`'s `__version__` string (which the rc1 stamp had missed, causing `health_check()` to advertise `1.6.5` to every client).

## The bug, plainly

Every MCP call that returned a row containing a datetime field was rejected at the MCP layer:

```
MCP error -32602: Structured content does not match the tool's output schema:
data/result/0/mtime must match format "date-time"
```

SQLite stores Curator's timestamps as naive strings like `'2026-05-10 09:08:10.516556'`. The MCP wire schema declares `format: date-time`, which demands RFC 3339 (T separator + timezone offset). Pydantic's default emission for a naive datetime fails both requirements.

## The fix

```python
def _as_utc(value: datetime) -> datetime:
    if value.tzinfo is None:
        return value.replace(tzinfo=timezone.utc)
    return value.astimezone(timezone.utc)

def _to_iso_utc_offset(value: datetime) -> str:
    return value.isoformat()  # produces +00:00 for aware UTC

UTCDatetime = Annotated[
    datetime,
    AfterValidator(_as_utc),
    PlainSerializer(_to_iso_utc_offset, return_type=str, when_used="json"),
]
```

Applied to: `SourceInfo.created_at`, `AuditEvent.occurred_at`, `FileSummary.mtime`, `TrashedFile.trashed_at`, `MigrationJobInfo.started_at`, `MigrationJobInfo.completed_at`.

## Breaking changes

**None.** The wire format change is from "invalid/rejected" to "valid RFC 3339 with `+00:00` offset" — clients that were getting schema-validation errors before now get usable data. No client could have been depending on the broken form because the broken form never reached them.
