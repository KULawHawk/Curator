## Lesson captured

No new lesson, but a synthesis worth recording: **two distinct flavors of unreachable defensive code** now handled in this arc:

- **Statement-level**: a `return None` or `raise` line that can't be reached → refactor (remove).
- **Branch-level**: a control-flow branch the language produces but the logic can't traverse → `# pragma: no branch` with justification.

Both honor "untested code is untrusted code" (doctrine item 1).
