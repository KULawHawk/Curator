## Lesson captured

No new lesson, but worth noting carry-forward: **when production code uses lazy imports inside method bodies** (`from PIL import Image` inside `_strip_image`), patching `service_module.Image.open` fails with AttributeError. Patch at the package-of-origin instead (`PIL.Image.open`, `pypdf.PdfReader`). Python-import-mechanics 101 but easy to forget at first attempt.
