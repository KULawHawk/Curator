## TL;DR

Ten ships of hygiene infrastructure (v1.7.65-v1.7.74) added scripts, hooks, lints, and Dependabot — but their existence was only documented inline in each file's header. **This ship adds a top-level "Contributing — dev setup" section to README.md** consolidating everything contributors need to know.

## What's in the new section

The new section (placed between `Install` and `Quick start`) covers:

- **One-command setup**: `.\scripts\setup_dev_hooks.ps1`
- **Three things the installer does** — hooksPath, PAT, verification
- **Pre-commit lints table** — glyph (v1.7.32), ORDER BY (v1.7.72), inline ANSI regex (v1.7.73) with scope + what each catches
- **Pre-push hook behavior** — state→action table (success/failure/in_progress/other)
- **CI diagnostic loop** — `ci_diag.ps1` modes with sample commands
- **Automated dependency tracking** — Dependabot scope and cadence
- **Inline exemption syntax** — `# order-by-lint:` and `# ansi-lint:` namespaces
- **Bypass commands** — `git commit --no-verify` and `git push --no-verify` for emergencies
