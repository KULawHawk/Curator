## What's worth noting

Three patterns from today that I'd flag for future Claudes:

1. **Honest "no new lesson" entries are fine.** Six of the eight ships today captured no fresh lesson. Each one logged that honestly. Fabricating thin lessons would corrupt the lesson library; honesty preserves its signal.

2. **Split discipline matters.** v1.7.93 became v1.7.93a + v1.7.93b at the natural API seam between progress-side helpers and persistent-job machinery. That decision (made early, per Lesson #88) prevented a mid-ship session end. Same call should apply to any future arc where a sub-ship grows beyond 1.5x typical.

3. **Coverage diff before shipping a test-file rewrite (Lesson #93).** The v1.7.92 ship surfaced this lesson when the autostrip test rewrite silently uncovered line 830. Every Coverage Sweep ship since has implicitly applied it — re-measure baseline at each ship start; verify the missing-line list shrinks; ship only when 100% is confirmed.
