## Key fixtures

- **`silence_qmessagebox`** — monkeypatches `QMessageBox.{about, critical, warning, information, question}` to `MagicMock`s. Tests can assert on calls without dialogs popping.
- **`stub_dialog_factory`** — given a dialog class name, replaces `curator.gui.dialogs.<name>` with a stub recording `.exec()`. Reused across all 7 Tools-menu happy-path tests.
- **`lineage_window`** — alternate fixture with populated time range so the slider is enabled (most slider-slot tests need this).

## Pattern: parametrize uniform slot families

The 7 `_slot_open_*` slots share the exact same shape (try-import / construct / exec / except → critical). A pair of `@pytest.mark.parametrize` decorators (one for happy, one for error) collapses 14 cases into 2 method definitions. Saves ~100 lines vs explicit per-slot tests and keeps the family's uniformity visible.
